# ECCV
The electrochemical commercial vehicle (ECCV) platform for Matlab Simulink is a free and open source environment for battery and fuel cell hybrid truck simulation. We only ask that you cite the original paper (
https://doi.org/10.3390/en17071742) if you use it for a publication.

# Quick-start
1. Download the files.
2. Add all folders to the MATLAB path.
3. Make sure your MATLAB working directory is the ECCV folder.
4. Run examples EX001 to EX005 in energies_2024-17-1742.
5. Use the simulink data inspector (SDI) to view simulation results.
6. Scripts to extract data from the SDI are provided in the post_process folder.
